========================================
OpenAI-Translator v2.0 启动说明
========================================

✅ 当前状态：
- Python 3.13.7 已安装
- Gradio 5.49.1 已安装
- 应用已配置完成

🚀 启动方式：

方式一：测试界面（推荐先测试）
  双击运行: start_simple.bat
  或命令行: python test_gradio.py

方式二：完整应用
  双击运行: start_app.bat
  或命令行: python main.py

📍 访问地址：
  http://localhost:7860

⚠️  注意事项：
1. 首次运行完整应用需要下载 ChatGLM2-6B 模型（约 12GB）
2. 如果端口 7860 被占用，可以修改 config.py 中的端口号
3. 如果无法访问，检查防火墙设置

📋 快速检查：
- 检查端口: netstat -ano | findstr :7860
- 查看进程: tasklist | findstr python
- 测试 Gradio: python -c "import gradio; print('OK')"
