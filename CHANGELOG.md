# 更新日志

## v2.0.0 (完善版)

### 新增功能

1. **进度显示**
   - PDF 翻译时显示实时进度
   - 显示当前翻译的页数和总页数

2. **PDF 信息显示**
   - 上传 PDF 后自动显示文件信息
   - 显示文件名、页数、文件大小

3. **语言选择优化**
   - 将文本输入改为下拉选择
   - 支持快速选择常用语言组合
   - 添加语言示例

4. **系统状态显示**
   - 显示翻译器和 PDF 处理器状态
   - 显示使用的设备（GPU/CPU）

5. **PDF 处理优化**
   - 支持长文本自动分段翻译
   - 改进 PDF 生成，更好的布局保留
   - 自动换行处理
   - 错误处理和回退机制

6. **翻译提示词优化**
   - 更详细的翻译要求
   - 更好的格式保留说明

7. **工具函数**
   - 添加文件大小格式化
   - 添加 PDF 信息获取
   - 添加语言验证

### 改进

1. **界面优化**
   - 更好的状态显示
   - 更清晰的信息展示
   - 改进的错误提示

2. **性能优化**
   - 长文本分段处理
   - 更好的内存管理

3. **错误处理**
   - 更完善的异常处理
   - 更友好的错误提示
   - PDF 生成失败时的回退机制

### 修复

1. 修复 PDF 生成时的布局问题
2. 修复长文本翻译时的截断问题
3. 改进特殊字符处理

## v1.0.0 (初始版本)

- 基础 PDF 翻译功能
- 文本翻译功能
- Gradio 图形界面
- ChatGLM2-6B 集成
